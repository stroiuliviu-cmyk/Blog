Despre profesie

Web Developer construiește interfețe și aplicații care rezolvă probleme reale. În proiectul meu: frontend-ul livrează design și validare, iar backend-ul oferă API și salvare date (JSON), ca să fie proiectul demonstrabil.

Ce învăț acum

Construiesc pagini responsive, creez formulare cu validare în JS și conectez UI-ul la un backend Node/Express care salvează mesajele în data/messages.json.

  • Responsive design
  • Validare (frontend + backend)
  • Fetch API către server
  • Structură clară de proiect

De ce IT?

În IT transform cunoștințele în produse reale: interfețe clare, logică corectă și feedback rapid.
